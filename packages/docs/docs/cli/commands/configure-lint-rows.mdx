# configure-lint-rows

configure the lint rule for rows of databases in the workspace

**dependencies**

```typescript
config.workspace.selectedPages.length > 0;
```

**arguments**

| name          | long  | short | prompt                  | enum            |
| ------------- | ----- | ----- | ----------------------- | --------------- |
| use lint rule | --use | -u    | select lint rule to use | ROWS_LINT_RULES |

**cli example**

```sh title="terminal"
npx mn configure-lint-rows --use lowercasePageTitles
```

**prompt example**

```sh title="terminal"
npx mn configure-lint-rows
```

```sh
select lint rule to use:
[x] lowercase page titles
[ ] untitled pages default to animal color names
[ ] pages without icons default to database icon
```
