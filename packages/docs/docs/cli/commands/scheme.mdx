# scheme

applies schematics code generation

**dependencies**

none

**arguments**

| name         | long           | short | prompt                                | default value |
| ------------ | -------------- | ----- | ------------------------------------- | ------------- |
| clear schema | --clear-schema | -c    | remove the existing schemad workspace | false         |

**cli example**

```sh title="terminal"
npx mn scheme --clear-schema
```

**prompt example**

```sh title="terminal"
npx mn scheme
```

```
remove the existing schemad workspace?
[ ] yes
[ ] no
```
