# configure-workspace

configures mount notion workspace

**dependencies**

```typescript
schema.length > 1;
```

**arguments**

| name            | long              | short | prompt                    | enum                  |
| --------------- | ----------------- | ----- | ------------------------- | --------------------- |
| schema package  | --schema          | -e    | name of schema package    |                       |
| api base url    | --base-url        | -b    | base url for api          |                       |
| auth strategies | --auth-strategies | -a    | authentication strategies | email, sms            |
| users database  | --users-database  | -d    | users database            | databases from schema |
| user column     | --user-column     | -c    | user column               |                       |

**cli example**

```sh title="terminal"
npx mn configure-workspace --schema @myth-software/schema -b https://api.myth.software -a sms -a email --users-database users --user-column owner
```

```sh title="terminal"
npx mn configure-workspace
```
