# auth-integration-key

authorization using an integration key generated in notion by a developer for internal integrations

**dependencies**

none

**arguments**

| name            | long              | short | prompt                 | default value |
| --------------- | ----------------- | ----- | ---------------------- | ------------- |
| integration key | --integration-key | -i    | notion integration key | none          |

**cli example**

```sh title="terminal"
npx mn auth-integration-key -i dcc1a640-95b6-49f3-bcc7-939ba90c3764
```

**prompt example**

```sh title="terminal"
npx mn auth-integration-key
```

```sh
notion integration key:
> secret_BO5Dah7xjfdafdajklfdajkl323CD
```
