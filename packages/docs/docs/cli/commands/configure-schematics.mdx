# configure-schematics

configures schematics to apply in configuration

**dependencies**

none

**arguments**

| name              | long           | short | prompt                                   | enum               |
| ----------------- | -------------- | ----- | ---------------------------------------- | ------------------ |
| schematics        | --schematics   | -s    | schematics to scheme                     | all schematics     |
| excludes from all | --excludes-all | -e    | databases to exclude from all schematics | databases in cache |

**cli example**

```sh title="terminal"
npx mn configure-schematics -s mirage -e logs
```

**prompt example**

```sh title="terminal"
npx mn configure-schematics
```

```sh
caches schematic is required, schemed automatially

schematics to scheme:
[ ] controllers
[ ] interfaces
[x] mirage
[ ] react-query
[ ] rtk-query
[x] locals

databases to exclude from all schematics:
[x] logs
[ ] sets
[ ] overlays

databases to exclude from mirage:
[ ] overlays
[x] sets

databases to exclude from locals:
[x] overlays
[x] sets
```
