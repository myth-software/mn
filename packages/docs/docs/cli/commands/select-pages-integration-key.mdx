# select-pages-integration-key

select pages manually, where there is no known list of available options

**dependencies**

```tsx
config.workspace.auth.strategy === 'integration-key';
```

**validation**

```tsx
isUuid(pageId);
isAvailable(pageId);
```

**arguments**

| name    | long      | short | prompt                                                        | default value |
| ------- | --------- | ----- | ------------------------------------------------------------- | ------------- |
| page id | â€”-page-id | -id   | page_id of selected page to include [press 'c' when complete] | none          |

**cli example**

```sh title="terminal"
mountn select-pages-integration-key --page-id=dcc1a64095b6-49f3-bcc7-939b-a90c3764
```

**prompt example**

```sh title="terminal"
npx mn select-pages-integration-key
```
