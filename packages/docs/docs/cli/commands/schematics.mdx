# schematics

applies schematics code generation

**dependencies**

none

**arguments**

| name        | long          | short | prompt                               | default value |
| ----------- | ------------- | ----- | ------------------------------------ | ------------- |
| clear cache | --clear-cache | -c    | remove the existing cached workspace | false         |

**cli example**

```sh title="terminal"
npx mn schematics --clear-cache
```

**prompt example**

```sh title="terminal"
npx mn schematics
```

```
remove the existing cached workspace?
[ ] yes
[ ] no
```
