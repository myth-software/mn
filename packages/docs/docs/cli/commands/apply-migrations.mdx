# apply-migrations

applies migrations for production drizzle databases

**dependencies**

```typescript
config.workspace.schematics.some((schematic) => schematic.name === 'drizzle');
```

**arguments**

| name    | long      | short | prompt         | default value | enum     |
| ------- | --------- | ----- | -------------- | ------------- | -------- |
| command | --command | -c    | select command |               | commands |

**cli example**

```sh title="terminal"
npx mn apply-migrations --command=generate
```

**prompt example**

```sh title="terminal"
npx mn apply-migrations
```

```sh
select command:
[ ] check
[ ] drop
[x] generate
[ ] migrate
[ ] up
```
