# lint-rows

lint workspaces’s databases rows for pass or fail against lint rules

**dependencies**

```typescript
config.workspace.lint.rows.length > 0;
```

**arguments**

none

**cli example**

```sh title="terminal"
3 databases rows to lint: 🔢 sets, 🔵 overlays, 📝 logs
```

```sh
pass 🔢 sets has lowercase page titles
pass 🔢 sets has untitled pages default to animal color names
pass 🔢 sets has pages without icons default to database icon
pass 🔵 overlays has lowercase page titles
pass 🔵 overlays has untitled pages default to animal color names
pass 🔵 overlays has pages without icons default to database icon
fail 📝 logs page_id 'bd7beed3-ba4a-499b-8f9d-16a4dd73e24f' has lowercase page titles
fail 📝 logs page_id 'bd7beed3-ba4a-499b-8f9d-16a4dd73e24f' has untitled pages default to animal color names
fail 📝 logs page_id 'bd7beed3-ba4a-499b-8f9d-16a4dd73e24f' has pages without icons default to database icon
```
