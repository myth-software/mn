# camp-interactive

interactive version of [camp](/cli/awesome-commands/camp)

**dependencies**

none

**arguments**

none

**cli example**

```sh title="terminal"
npx mn camp-interactive
```

```sh
camping...

press

1 - configure-workspace
2 - configure-lint-columns
3 - configure-lint-rows
4 - configure-schematics
5 - inspect the entities
6 - inspect the configuration file
7 - fetch, build, scheme

> 5
opening entities in code editor

> 6
opening configuration file in code editor
```
