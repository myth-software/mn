# camp

runs a watch command then calls an api on press of keyboard

**dependencies**

none

**arguments**

| name        | full          | short | promprt                              | default |
| ----------- | ------------- | ----- | ------------------------------------ | ------- |
| clear cache | --clear-cache | -c    | remove the existing cached workspace | false   |
| fetch       | --fetch       | -f    | run the fetch command                | true    |
| skip-lint   | --skip-lint   | -s    | skip lint rules                      | false   |

**cli example**

```sh title="terminal"
npx mn --clear-cache --fetch false --skip-lint
```

**prompt example**

```sh title="terminal"
npx mn camp
```

```sh
remove the existing cached workspace:
[ ] yes
[ ] no

options
- cleanup will clear the cache
- fetch will refetch all the notion data
- scheme will rescheme all the schematics

mn configure

options
- manual will open the editor to the json configuration
- inspect will open the editor to the root entities configuration

```

**stages**

| name             | condition                                              |
| ---------------- | ------------------------------------------------------ |
| cleanup          | if climber chooses to clear cache                      |
| fetch            | if there is no cache                                   |
| lint-rows        | if there is configure-lint-rows and not --skip-lint    |
| lint-columns     | if there is configure-lint-columns and not --skip-lint |
| fix-rows         | if failing rows and not --skip-lint                    |
| fix-columns      | if failing columns and not --skip-lint                 |
| apply-schematics | if the cache has changes                               |
