## getting started

### what you'll need

- [nx monorepo](https://nx.dev//) version 16 or above
- [notion account](https://affiliate.notion.so/xysfb04gkht8) (affiliate link)

### installation

```sh title="terminal"
npm i --save-dev @mountnotion/{cli,types,schematics,sdk,utils}
```

### running

```sh title="terminal"
npx mn schematics
```
